# Tests

## Backend

- `npm run test` : exécute les tests Vitest.
- `npm run test:coverage` : lance Vitest avec couverture.
- Les tests se trouvent dans `backend/src/__test__/`.

## Backoffice

- `npm run test` : exécute les tests Vitest.
- `npm run test:coverage` : lance Vitest avec couverture.
- `npm run test:e2e` : exécute Playwright.
- Les tests se trouvent dans `backoffice/src/__test__/`.

## User-front

- `npm run test` : exécute les tests Vitest.
- `npm run test:coverage` : lance Vitest avec couverture.
- Les tests se trouvent dans `user-front/src/__test__/`.

## Stratégie

- tests unitaires pour le backend, le front office et le client mobile
- tests end-to-end pour le backoffice avec Playwright
